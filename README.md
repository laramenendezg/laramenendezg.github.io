# laramenendezg.github.io

Personal website / CV for Lara Menendez.

This repo contains the static site built with a downloaded template. The site is published via GitHub Pages at:

https://laramenendezg.github.io

To update the site:
- Clone this repo
- Edit files (index.html, css/, js/, assets/)
- Commit & push to the default branch (main)

If you want a custom domain, add a `CNAME` file containing your domain and configure DNS as described in GitHub Pages docs.
